# Style by Language

> Vom `localization-coordinator` gepflegt. Tonfall pro Sprache.

## DE

- Comic-haft überdreht, mit zwinkerndem Auge — siehe lore-writer/tone-of-voice.md
- Du-Form (nicht Sie)
- Kurze Sätze in Tooltips, max 2 Zeilen
- Pseudowissenschaftliche Begriffe sind erlaubt und gewünscht
- Vermeiden: Anglizismen wo es deutsche Varianten gibt
  (Ausnahme: gaming-Begriffe wie „Run", „Boss", „Build")

## EN

- Same humor as DE — wisecracks, paleontology nerd jokes
- Sentence-case in UI labels (not Title Case)
- Avoid jargon that doesn't translate from DE feel
