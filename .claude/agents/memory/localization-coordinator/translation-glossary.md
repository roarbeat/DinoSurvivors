# Translation Glossary

> Vom `localization-coordinator` gepflegt. Begriffs-Konsistenz pro Sprache.

## DE-Glossary

| EN | DE | Anmerkung |
|----|----|----|
| amber (currency) | Bernstein | NICHT „Amber" — DE-Spieler erwarten Bernstein |
| boss | Boss | nicht „Endgegner", Boss ist im DE-Gaming-Sprech etabliert |
| crit | Krit | „kritisch" für Tooltips, „Krit" als Token in UI |
| damage | Schaden | nicht „DMG", außer in Compact-UI |
| level up | Stufenaufstieg | bei Run-Level; „Aufstieg" für Meta-Progression |
| mutation | Mutation | identisch |
| run | Run | bewusster Anglizismus, im Kontext eindeutiger als „Lauf" |
| wave | Welle | identisch |
| xp | EP | im DE-Sprachraum etabliert |
