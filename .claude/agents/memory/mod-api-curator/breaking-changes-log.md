# Breaking Changes Log

> Vom `mod-api-curator` gepflegt. Historie aller Breaking Changes mit
> Begründung und Migration-Guide.

> **Regel:** Jeder Eintrag muss enthalten:
> 1. Welche Version führt das ein
> 2. Was genau bricht
> 3. Warum unvermeidbar
> 4. Migration-Schritt für Mod-Authoren

---

## (noch keine Breaking Changes)

v0.0.1 ist der initiale Public-API-Snapshot. Alle nachfolgenden Versionen
werden gegen v0.0.1 verglichen.

---

## Format-Vorschlag für künftige Einträge

### v0.X.0 — `<Komponente>` `<Operation>`

- **Eingeführt in:** v0.X.0 (Datum)
- **Bricht:** ...
- **Warum:** ...
- **Vermeidbar?** ja / nein — falls nein: Begründung
- **Migration:**
  ```
  // Vor v0.X.0
  ...
  // Ab v0.X.0
  ...
  ```
- **Deprecation-Warning seit:** v0.X-1.0 (falls vorab gewarnt)
