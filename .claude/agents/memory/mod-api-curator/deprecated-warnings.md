# Deprecated Warnings

> Vom `mod-api-curator` gepflegt. Was wir gerade deprecaten — Mod-Authoren
> bekommen Push-Warnings im Output, damit sie umstellen können bevor das
> nächste Major-Update bricht.

---

## (aktuell keine Deprecations)

## Format-Vorschlag

### `<API-Element>`

- **Deprecated seit:** v0.X.0 (Datum)
- **Wird entfernt:** v0.Y.0 (geplant) oder „kein Termin"
- **Stattdessen:** ...
- **Warning-Code:** „DEPR-XXX" (für Log-Filter)
