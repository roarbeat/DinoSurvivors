# art/ui/

> UI-Assets: Pixel-Font, 9-Slice-Frames, HUD-Icons.

## Erwartete Files

| File | Inhalt |
|------|--------|
| `frame_default.png` | 9-Slice-Frame für MutationPickOverlay etc. |
| `pixel_font.tres` | BitmapFont-Resource (m5x7, PixelOperator, etc.) |
| `icon_amber.png` | Bernstein-Icon für HUD-Currency-Display |
| `icon_health.png` | Herz-Icon für HP-Bar |
| `icon_wave.png` | Wellen-Icon für HUD-Wave-Counter |

## Font-Convention

Pixel-Font sollte 7-9 Pixel hoch sein, damit Damage-Numbers + HUD-Texte
zum Stil der Sprites passen. AAA-Smooth-Fonts brechen den Look.
